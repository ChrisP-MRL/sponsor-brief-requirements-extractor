Color Usage Guidelines

  • Primary Red (#CE372F): Reserve for headlines, key callouts, important buttons, and brand emphasis. Never use as body text background.
  • Dark Teal (#00444C): Excellent for footers, secondary navigation, and professional dark sections.
  • Black on White: Default for body copy to ensure maximum readability.
  • Shade Numbering System: Each accent color family has 5 shades (100-500). Lower numbers (100-200) are lighter for backgrounds and subtle highlights. Mid-range (300) for icons and moderate emphasis. Higher numbers (400-500) for text on light backgrounds and strong accents.
  • Background Colors: Use 100-200 shades for backgrounds and light sections (Brown 100 #F1EDE7, Grey 100 #E8E8E8, Cyan 100 #E0F7F9).
  • Accent Colors for Data Viz: Use 300-400 shades for charts, graphs, and data visualizations. The mid-range provides good contrast and visual distinction.
  • Color Families by Purpose: Grey scale for neutral elements; Wine/Burgundy for deep accents; Cyan for technical/digital themes; Coral/Red for primary emphasis; Brown for earth/natural themes; Terra Cotta for warm highlights; Olive for sustainability/environmental; Teal for professional/corporate.
  • Limit Per Design: Use 2-3 accent color families per design to maintain visual cohesion.
  • Contrast Ratio: Maintain WCAG AA compliance with minimum 4.5:1 contrast ratio for body text, 3:1 for large text. Lighter shades (100-200) should use dark text; darker shades (400-500) should use white text.

2. Typography Standards

Primary Typeface: Century Gothic

Century Gothic is the official MinRes typeface for both headings and body copy. Its clean, modern geometric design conveys professionalism and clarity.

Heading 1 - Century Gothic Bold

Font: Century Gothic Bold | Size: 32-40pt | Color: #CE372F | Use: Page titles, major section headers

Heading 2 - Century Gothic Bold

Font: Century Gothic Bold | Size: 24-28pt | Color: #00444C | Use: Section titles, subheaders

Heading 3 - Century Gothic Semibold

Font: Century Gothic Semibold | Size: 18-22pt | Color: #000000 | Use: Subsection headers

This is body copy at 15pt. Use Century Gothic Regular for all body text. Maintain line spacing of 1.8 for optimal readability. Ensure sufficient paragraph spacing (12pt after) for comfortable reading flow.

Font: Century Gothic Regular | Size: 15pt | Color: #000000 | Line Height: 1.8 | Use: Body text, descriptions, content

Emphasis Text - Bold for Important Points

Font: Century Gothic Bold | Size: 15pt | Color: #CE372F | Use: Emphasis, key terms, important callouts

Alternative: Avenir

When Century Gothic is unavailable, use Avenir as the secondary typeface. Avenir Black for headings, Avenir Regular for body copy.

Typography Rules

  • Hierarchy: Maintain clear visual hierarchy with size and weight. Headlines should be bold and prominent.
  • Body Text Size: Always use 15pt for body copy. Never go below 12pt for any text.
  • Line Spacing: Set line height to 1.8 (180%) for body text to ensure comfortable reading.
  • Paragraph Spacing: Use 12pt spacing after paragraphs instead of indents.
  • ALL CAPS: Use sparingly and only for short headings or labels. Never use for body text.
  • Text Color: Primary text should be black (#000000) on white backgrounds. Use MinRes Red (#CE372F) for emphasis only.
  • Font Weights: Bold (700) for headings, Semibold (600) for subheadings, Regular (400) for body.

Icon Guidelines

  • Style: Use line-based icons with 2-3pt stroke weight. Solid fills acceptable for simple shapes.
  • Format: Prefer SVG format for scalability. PNG at minimum 512x512px if raster required.
  • Color: Icons should use brand colors. Primary Red for active/important, Dark Teal for standard, accent colors for categories.
  • Size: Design icons on 24x24px or 48x48px grid. Maintain consistent sizing across sets.
  • Simplicity: Icons must be clear and recognizable at small sizes (16px minimum).
  • Consistency: Maintain consistent stroke weight, corner radius, and style across all icons in a set.
  • Industry Relevance: Use industry-appropriate iconography (mining equipment, safety, environment, logistics).

Visual Element Placement

  • Logo Placement: MinRes logo should appear in top-left or bottom-left. Maintain clear space equal to logo height.
  • Image Treatment: Use high-quality photography. Apply subtle overlays if text overlay required (rgba(0,0,0,0.3)).
  • Graphics: Keep graphics clean and purposeful. Avoid decorative elements that don't add meaning.
  • White Space: Embrace white space for breathing room. Don't overcrowd layouts.

4. Layout & Composition Standards

Document Structure Example

DOCUMENT TITLE OR SECTION HEADER

Section Heading

Body content goes here at 15pt Century Gothic Regular. Maintain generous spacing between elements for clarity and professionalism. Use the warm beige background (#F1EDE7) for alternating sections to create visual rhythm without overwhelming the design.

Key Point: Emphasized text uses bold weight and MinRes Red color.

FOOTER: PRESENTATION NAME | PAGE NUMBER | © Mineral Resources Limited

Layout Principles

  • Grid System: Use 12-column grid for web, flexible margins for documents. Maintain alignment and consistency.
  • Margins: Minimum 40px margins on all sides for digital content. 1 inch margins for print documents.
  • Spacing: Use consistent spacing scale (8px, 16px, 24px, 32px, 40px) for element separation.
  • Headers: Place headers at top with MinRes Red background or as left-aligned with red accent bar.
  • Footers: Include presentation name, page number, and branding in footer. Use Dark Teal background.
  • Content Blocks: Break long content into digestible sections. Use the warm beige (#F1EDE7) for alternating backgrounds.
  • Asymmetry: While alignment is key, asymmetric layouts can create visual interest. Balance is more important than symmetry.

Responsive Considerations

  • Mobile-First: Design for smallest screen first, then scale up.
  • Typography Scale: Reduce heading sizes by 20-30% on mobile while maintaining hierarchy.
  • Touch Targets: Minimum 44x44px for interactive elements on mobile.
  • Single Column: Stack content in single column on narrow viewports.

7. Best Practices & Common Scenarios

Do's and Don'ts

✓ DO

  • ✓ Use MinRes Red (#CE372F) for emphasis and key callouts
  • ✓ Maintain consistent spacing and alignment
  • ✓ Use high-quality, professional imagery
  • ✓ Keep layouts clean with generous white space
  • ✓ Use Century Gothic or Avenir exclusively
  • ✓ Ensure WCAG AA contrast compliance
  • ✓ Test content at multiple screen sizes
  • ✓ Use brand accent colors purposefully

✗ DON'T

  • ✗ Mix multiple typefaces beyond brand fonts
  • ✗ Use colors outside the defined palette
  • ✗ Overcrowd layouts with too many elements
  • ✗ Use font sizes below 12pt for any text
  • ✗ Apply excessive effects (gradients, shadows, glows)
  • ✗ Ignore accessibility requirements
  • ✗ Use low-quality or generic stock images
  • ✗ Create inconsistent icon styles

Accessibility Checklist

  • Color Contrast: Verify all text meets WCAG AA standards (4.5:1 for normal text, 3:1 for large text >18pt).
  • Alternative Text: Provide descriptive alt text for all images and icons in AI-generated content.
  • Font Size: Never use text smaller than 12pt. Prefer 15pt for body content.
  • Color Alone: Don't rely solely on color to convey information. Use text labels, icons, or patterns.
  • Focus Indicators: Ensure interactive elements have clear focus states for keyboard navigation.
  • Readability: Maintain line length of 50-75 characters for optimal readability.
